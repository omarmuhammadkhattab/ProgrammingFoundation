Loops in Python

Stop writing repetitive code — use loops to repeat actions efficiently. Master while loops, for loops, nested loops, and loop control statements.

while loop for loop range() nested loops break continue pass
🔁

Why Do We Need Loops?

Imagine printing numbers from 1 to 100. Without loops, you'd need 100 print() statements. With a loop — just 2 lines. Loops let you repeat actions efficiently.

while loop
Repeats a block of code as long as a condition is True. Use when you don't know in advance how many times to repeat.
for loop
Iterates over a sequence (list, string, range, etc.). Use when you know what you're iterating over.
nested loops
A loop inside another loop. The inner loop completes fully for every single iteration of the outer loop.
💡 Key difference: A while loop checks a condition first, then executes. A for loop steps through items in a sequence one by one.
📐

The range() Function

range() is a built-in Python function that generates a sequence of integers. It's most commonly used with for loops to control how many times the loop runs.

SyntaxGeneratesExample output
range(5)0 up to (not including) 50, 1, 2, 3, 4
range(1, 7)1 up to (not including) 71, 2, 3, 4, 5, 6
range(1, 20, 2)1 to 20, step by 21, 3, 5 … 19
range(20, 0, -2)20 down to 2, step by -220, 18, 16 … 2
range() examples
# Count from 0 to 4 for i in range(5): print(i) # 0 1 2 3 4 # Count from 1 to 6 for i in range(1, 7): print(i) # 1 2 3 4 5 6 # Odd numbers: step by 2 for i in range(1, 20, 2): print(i, end=" ") # 1 3 5 7 … 19 # Count backwards for i in range(20, 0, -2): print(i, end=" ") # 20 18 16 … 2

The while Loop

A while loop keeps executing its block of code as long as the condition remains True. The moment the condition becomes False, the loop stops and the program moves on.

while loop syntax & examples
while expression: statement(s) # Example 1: countdown count = 5 while count > 0: print(count) count -= 1 # IMPORTANT: update the variable! print("Liftoff!") # Example 2: sum until limit total = 0 n = 1 while total < 100: total += n n += 1 print(f"Reached {total} after adding {n-1} numbers")
⚠️ Infinite loop danger! If your condition never becomes False, the loop runs forever. Always make sure the loop variable is updated inside the loop — otherwise your program will freeze.
# ❌ INFINITE — count never changes!
count = 5
while count > 0:
    print(count) ← missing count -= 1

The for Loop

For loops are used for sequential traversal — stepping through each item in a sequence (list, tuple, string, dictionary, range) one by one.

for loop syntax & examples
for iterator_var in sequence: statements(s) # Iterating over a list fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(f"I like {fruit}") # Iterating over a string for char in "Python": print(char, end=" ") # P y t h o n # Iterating with range for i in range(1, 6): print(f"{i} x {i} = {i*i}") # Iterating over a dictionary student = {"name": "Ahmed", "grade": "A"} for key, val in student.items(): print(f"{key}: {val}")
🔑 Choosing between for and while: Use for when you know what to loop over (a list, a range). Use while when you loop based on a condition that changes.
🪆

Nested Loops

Python allows placing one loop inside another. The inner loop runs completely for every single iteration of the outer loop. Useful for grids, matrices, and combinations.

nested for & nested while syntax
# Nested for loop for iterator_var in sequence: for iterator_var in sequence: statements(s) statements(s) # Nested while loop while expression: while expression: statement(s) statement(s)
nested loops — multiplication table
# Print a 3x3 multiplication table for i in range(1, 4): for j in range(1, 4): print(i * j, end="\t") print() # Output: # 1 2 3 # 2 4 6 # 3 6 9 # Star pattern for i in range(1, 6): print("* " * i)
🧠 Execution order: For each single step of the outer loop, the inner loop runs all the way through. A 3×4 nested loop runs the inner body 3 × 4 = 12 times total.
🎛️

Loop Control Statements

Sometimes you need to change a loop's normal flow. Python provides three loop control statements:

break
Immediately ends the loop entirely. Execution jumps to the first statement after the loop.
continue
Skips the rest of the current iteration and jumps straight to the next one.
pass
Does nothing — a placeholder so you can write the body later without a syntax error.
break — stop the loop early
for i in range(1, 10): if i == 5: print("Found 5! Stopping.") break print(i) # Output: 1 2 3 4 Found 5! Stopping.
continue — skip this iteration
for i in range(1, 10): if i % 2 == 0: continue # skip even numbers print(i) # Output: 1 3 5 7 9
pass — placeholder
for i in range(5): pass # TODO: add logic here print("Loop completed (pass did nothing)")
