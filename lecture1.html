What is Programming?

A program is simply a list of instructions that a computer follows to accomplish a task. The person who writes those instructions is called a programmer, and the activity of writing them is called programming.

Computers only understand machine language (0s and 1s). Programming languages like Python act as a human-friendly bridge — they get translated into machine language by a special program called a compiler or interpreter.

🐍

What is Python?

Python is a popular, general-purpose programming language created by Guido van Rossum and first released in 1991. It's designed to be easy to read and write, making it perfect for beginners and professionals alike.

Python Comments

Comments are notes for humans, not the computer. Python completely ignores them. They help explain your code and make it easier to maintain.

single-line comment
# This is a comment — Python ignores this line print("Hello, World!") # inline comment # Comments can also disable code during testing: # print("this line won't run") print("this line WILL run")
multi-line comment
""" This is a multi-line comment. You can write across many lines. It uses triple double-quotes. """ print("Code resumes here")
📦

Variables in Python

A variable is a named container that stores a value in memory. Unlike many other languages, Python doesn't need you to declare the type — just assign and go!

Dynamically typed: In Python, a variable can change its type. x = 5 makes x an integer, then x = "hello" makes x a string — no problem!
📝

Variable Naming Rules

Variable names can contain letters (A-Z, a-z), digits (0-9), and underscores (_). The first character must NOT be a digit. Names are case-sensitive.

Reading Input & Printing Output

Python uses input() to read from the user and print() to display on the screen.

input and output
# input() always returns a string name = input("Enter your name: ") print("Hello, " + name) # Convert string to number with int() age = int(input("Enter your age: ")) print("You are", age, "years old") # f-strings are a clean way to format output print(f"Hello {name}, you are {age} years old!")
⚠️ Remember: input() always returns a string. If you need a number, wrap it: int(input(...)) or float(input(...))
