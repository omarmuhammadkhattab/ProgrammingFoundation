Basic Data Types

Every value in Python has a type. The type tells Python what kind of data it is and what operations make sense on it. The four basic types are:

int
Whole numbers — positive, negative, or zero. No decimal point.
x = 42
y = -7
z = 0
float
Numbers with a decimal point. Used for precise measurements.
pi = 3.14
temp = -2.5
price = 99.9
str
Text — any sequence of characters, wrapped in quotes.
name = "Ahmed"
city = 'Cairo'
msg = "Hello!"
bool
Only two values: True or False. Used in conditions.
is_student = True
passed = False
type checking & conversion
x = 42 print(type(x)) # <class 'int'> pi = 3.14 print(type(pi)) # <class 'float'> # Convert between types s = str(42) # "42" (string) n = int("100") # 100 (integer) f = float("3.5") # 3.5 (float)
🔤

Working with Strings

Strings are sequences of characters. Python has many built-in methods to manipulate them.

string methods
name = "Cairo University" print(name.lower()) # "cairo university" print(name.upper()) # "CAIRO UNIVERSITY" print(len(name)) # 16 print(name.islower()) # False print(name.isupper()) # False # String concatenation greeting = "Hello, " + name + "!" print(greeting)

slicing examples
s = "Python" print(s[0]) # 'P' — first character print(s[-1]) # 'n' — last character print(s[0:3]) # 'Pyt' — index 0, 1, 2 (end is excluded) print(s[:3]) # 'Pyt' — from start to index 2 print(s[3:]) # 'hon' — from index 3 to end print(s[::-1]) # 'nohtyP' — reversed!
