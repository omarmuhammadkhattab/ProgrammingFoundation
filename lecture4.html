Conditions & Branching

Control the flow of your program — make it take different paths depending on what conditions are true or false.

What is Branching?

Real programs don't just run top to bottom — they make decisions. Should we print "Pass" or "Fail"? Is the user logged in? Branching lets your code take different paths based on conditions.

if
Execute a block only when condition is True
elif
Check another condition if the first was False
else
Run when ALL previous conditions were False
nested
Put if statements inside other if statements
1️⃣

The if Statement

The simplest form. The block of code inside runs only if the condition is True. If the condition is False, the block is completely skipped.

if statement
if condition: # ↑ colon is required! block_of_code # ↑ 4-space indent required! # Example: temperature = 35 if temperature > 30: print("It's hot outside!") print("Drink water.") print("Program continues here") # always runs
🔑 Key syntax rules: Always end the condition with a : (colon). The code block MUST be indented — Python uses indentation to define blocks (4 spaces is standard).
2️⃣

if…else Statement

The else block provides a fallback — code that runs when the if condition is False. Exactly one of the two blocks will always run.

even or odd
number = int(input("Enter a number: ")) if number % 2 == 0: print(f"{number} is Even") else: print(f"{number} is Odd")
3️⃣

elif — Multiple Conditions

elif (short for "else if") lets you chain multiple conditions. Python checks them in order — as soon as one is True, it runs that block and skips the rest.

DS502 grade calculator
score = float(input("Enter your score: ")) if score >= 90: grade = "A" elif score >= 80: grade = "B" elif score >= 70: grade = "C" elif score >= 60: grade = "D" else: grade = "F" print(f"Your grade is: {grade}")

4️⃣

Nested if Statements

You can put an if statement inside another if statement. This lets you check conditions within conditions. Be careful with indentation — each level needs another 4 spaces.

nested if example
age = 20 has_id = True if age >= 18: print("Age is sufficient.") if has_id: # nested if print("Access granted!") else: print("Please show your ID.") else: print("Access denied — too young.")
⚠️

Common Errors

❌ Wrong indentation
Python uses indentation to define code blocks. Mixing spaces and tabs, or inconsistent spacing, causes IndentationError.
if x > 0:
print("wrong") ← not indented → ERROR!
❌ = instead of ==
Using = (assignment) instead of == (comparison) in conditions causes a SyntaxError.
if x = 5: ← assigns, doesn't compare → ERROR!
if x == 5: ← correct!
